The campaign object, which U get with GET campaign give U the data of the campaign table and 3 related tables (Location, CampaignTopics and PublishChannel).

Location is a table, where all regions in which campaigns could be take place.
Every campaign needs this information. Therefore the POST method needs a valid Location id. U need this:


<location>
<idLocation>1</idLocation>
<name>Germany</name>
</location>

OR


<location>
<idLocation>1</idLocation>
</location>

It is also possible to add some topics (in my understanding a list of tags) to the campaign:

<topics>
<topic>
<topic>Campaigntopic 1</topic>
</topic>
<topic>
<topic>Campaigntopic 2</topic>
</topic>
</topics>
